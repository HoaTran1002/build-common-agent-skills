# Claude Project Instructions

## Shared skills

This repository keeps portable agent workflows in `.agents/skills`.

When the user asks for PR review, debugging test failures, or release notes, inspect the matching skill:

- PR review: `.agents/skills/quality/pr-review/SKILL.md`
- Test failure debugging: `.agents/skills/debug-failing-test/SKILL.md`
- Release notes: `.agents/skills/quality/release-notes/SKILL.md`

Follow the skill instructions exactly.
Only read supporting files such as `checklist.md` or `template.md` when the skill asks for them.
